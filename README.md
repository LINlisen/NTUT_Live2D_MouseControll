# NTUT_Live2D_MouseControll
國立臺北科技大學-Live2D滑鼠跟隨

## 環境建置
### 下載檔案
右上方Code->Download ZIP
### 開啟檔案
1. 開啟Unity Hub
2. 專案->右上角開啟->選擇下載下來解壓縮的檔案

## 執行檔
* [GitHub](https://github.com/LINlisen/NTUT_Live2D_MouseControll_exe.git)
下載後開啟Live2D_MouseControll.exe即可看到Demo的樣子
